# RAG is All You Need
가짜 연구소 9기 "RAG is All You Need!" 스터디 페이지 입니다

## 프로젝트 소개
- 본 스터디는, RAG의 과거부터 현재까지 알아보는 스터디 입니다. 스터디에서는 RAG의 이해와 현재 최신 기술에 대해 깊게 알아보며, 비판적으로 RAG 기술들을 파악 및 분석하는 것이 목표입니다.
